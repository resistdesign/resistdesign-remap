remap

Create a new object or array by remapping the properties of the supplied value.

remap(value: (Object | Array), map: Object<string, string>, createArray: boolean): (Object | Array)
Parameters
value ((Object | Array)) The source value.
map (Object<string, string>) The map used to reassign values. Keys are the destination keys and values are the source keys.
createArray (boolean = false) A flag designating whether or not an array will be returned.
Returns
(Object | Array): The remapped value.
